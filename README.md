# Portfolio

This is my personal portfolio website.

I built this project to showcase my work in digital marketing, community leadership, and my growing skills in web development.

The site includes sections for:
- About me
- Projects and experiences
- LinkedIn content and event participation
- Contact form

## Built With

- React
- Vite
- JavaScript
- CSS

## Run Locally

Install dependencies:

npm install

Start the development server:

npm run dev
